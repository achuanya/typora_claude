- # Typora Lite

  一个为专注阅读、长文写作和 Markdown 编辑设计的 Typora 衬线主题
  
  本主题 Fork 自开源项目：[blaxisomu/typora_claude](https://github.com/blaxisomu/typora_claude)
  
  我在原版的基础上，进行了瘦身，尽量保证在写作的过程中不受主题的视觉干扰
  
  
  
  ## 使用方法
  
  1. `git clone` 项目源码，或点击`Code`按钮，选择`Download ZIP`
  
  2. 将根目录中的 `lite.css`、`lite-dark.css` 以及 `lite-fonts` 文件夹一起拖入 Typora 的主题文件夹中
  
  3. **如何打开主题文件夹？**
  
     1. 打开 Typora 软件中的 `偏好设置`
     2. 选择 `外观`，随后将页面滚动至底部
     3. 找到并点击 `打开主题文件夹` 按钮
  
  4. 重启 Typora，在顶部菜单栏的 `主题` 中选择 `Lite` 或 `Lite Dark` 即可
  
     
  
  ## Lite 版核心改动
  
  
  
  相比于原版主题，`Typora Lite` 做了以下瘦身：
  
  - **取消纲树状连线**
  - **取消目录、大纲高亮背景块**
  - **取消目录动画**
  - **取消目录艺术字**
  - **统一配色**
